A lightweight, dark-mode-first CSS framework built for speed, clarity, and experimentation.

Crafted for developers who want power without the noise — utility-first styling, clean bracket-based responsive syntax, and a anon-inspired design system that scales from tiny projects to production-ready interfaces.

Bracket-Based Responsive Syntax

Write cleaner responsive code with [ ] utilities instead of repetitive breakpoint prefixes. Reduce clutter and focus on intention — md-[p-4 text-center bg-accent] compiles into optimized, predictable CSS.

Dark-Mode-First Cyber Design System

A consistent, modern palette inspired by dark UI, neon accents, and cyber interfaces. Perfect for dashboards, tools, and sleek web apps — built on custom CSS variables that stay predictable and themeable.

Tiny, Utility-First Core

A fast, minimal utility engine that gives you the power of Tailwind without the bulk. Every class is handcrafted for clarity, extendability, and learning how frameworks work from the inside.
